1Summary of techniques used in molecular construction

  • Select Molecular libary

    • Determinictis method: Select pure component base on knownledge/ well analyzed sample

    • Stocastic method: Using Motecarlo simulation to select thousands of molecule stochasticly Using it when cannot analyze sample heavy fraction

  • Represent molecular libary

    • By pure component: When the molecular libary is small (10x ~ 100x)

    • By SOL (Structure oriented lump) method:

    • By MTHS (Molecular type homogulus series): Using when we don't consider isomer

  • Calculate pure properties

    • Properties from database (Aspen Plus, Nist, Dortmund database) suitable for deterministic method, pure component

    • Group constribution method suitable for stochastic method, MTHS, SOL

  • Calculate mixture properties

    • Non-linear mixing rules: Use with any bulk properties, can consider non-ideality suitable for regression method.

    • Linear mixing rules: Apply for some comercial indices (Distillation curve, Elemental analysis, PIONA analysis, density ...) suitable for maximizing entropy method.

  • Adjust molar fraction

    • Regression method: Adjust molarfraction to fit the calculated bulk properties with measured bulk properties. The molar fraction could be assumed follow some distribution ( distribution, Gaussiant distribution, ... ) to reduce the degree of freedom.

    • Maximizing entropy method: Use bulk properties as constrain and Shannon entropy function is objective function to optimize.

  • Transfer to the assay: Transfer result back to the molecular libary (pure component, SOL, MTHS ... )
